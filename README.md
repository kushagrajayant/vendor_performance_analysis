# 📊 Vendor Performance Analysis

## 📌 Project Overview

The **Vendor Performance Analysis** project is designed to evaluate and monitor vendor performance by analyzing procurement, sales, inventory, and financial data. The project combines **SQL** for data extraction, **Python** for data cleaning and exploratory analysis, and **Power BI** for interactive dashboards and business insights.

The objective is to help businesses identify top-performing vendors, optimize procurement strategies, reduce costs, and improve overall supply chain efficiency.

---

## 🚀 Technologies Used

- **Power BI** – Interactive dashboards and data visualization
- **SQL** – Data extraction, transformation, and querying
- **Python** – Data cleaning, preprocessing, and analysis
- **Pandas** – Data manipulation
- **NumPy** – Numerical computations
- **Matplotlib & Seaborn** – Data visualization
- **Jupyter Notebook** – Exploratory Data Analysis (EDA)

## 🗄 SQL Tasks

- Data extraction
- Data cleaning
- Joining multiple tables
- Aggregations
- Vendor-wise performance calculations
- Sales and purchase analysis
- Inventory analysis

### Sample SQL Query

```sql
SELECT
    VendorName,
    SUM(PurchaseAmount) AS TotalPurchases,
    SUM(SalesAmount) AS TotalSales
FROM VendorPerformance
GROUP BY VendorName
ORDER BY TotalSales DESC;
```
